# Theme Factory

Apply 10 curated themes to artifacts (slides, docs, HTML). Themes include: Ocean Depths, Sunset Boulevard, Forest Canopy, Modern Minimalist, Golden Hour, Arctic Frost, Desert Rose, Tech Innovation, Botanical Garden, Midnight Galaxy.

## Example Prompts

- "Apply the Ocean Depths theme to this presentation deck"
- "Restyle this HTML page using the Modern Minimalist theme"
- "Show me a preview of all available themes for my document"
- "Switch the current design from Golden Hour to Arctic Frost"
