# Bean Burgers

> The simplest bean patty recipe that actually works. Blend, fry, eat. If it falls apart, don't panic - just make Hecshia instead. Failures become new dishes here.

## Ingredients

- Kidney beans
- Onions
- Oats (put enough or it will fall apart!)
- Ketchup
- Spices (your choice)

## Instructions

1. Blend all ingredients together
2. Add enough oats to hold it together
3. Form into patties
4. Fry until golden on both sides

### If It Falls Apart (Hecshia)
1. Don't panic!
2. Crumble the mixture
3. Add diced potato (batata) and more onion
4. Fry it all up into a delicious hash
5. Call it Hecshia and pretend that was the plan all along
